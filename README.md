# Осознанные сны [в разработке]
Мобильное приложение для ведения дневника снов и практики метода тестирования реальности. Построено 
по принципу single-activity architecture с использованием паттерна MVVM.
</br>
Особенность проекта: симуляция стратегии ветвления GitFlow с помощью создания дополнительного
[гит-аккаунта](https://github.com/kirillpolonskiismurf/LucidDreaming), выполняющего роль второго
участника команды.</br>
Использованные рабочие процессы:
 - Pull request
 - (coming soon) Симуляция code review с принятием и отказом PR

## Стек
 - Kotlin
 - View-based UI
 - Jetpack Navigation
 - Room
 - Dagger Hilt
 - View binding
 - (coming soon) Coroutines & Flow
 - (coming soon) WorkManager
## Возможности приложения
 - (coming soon) Создание, редактирование снов, их поиск с помощью различных фильтров
 - (coming soon) Настройка сильных и слабых напоминаний (звук, текст, период активности, количество в день)
 - (coming soon) Выбор одной из 4 цветовых тем
 - (coming soon) Смена языка
## Реализованный функционал
 - Использование Navigation Component для переходов между фрагментами
 - (coming soon) Передача аргументов навигации с помощью плагина Safe Args
 - (coming soon) Создание фоновой повторяющейся задачи с помощью WorkManager
 - (coming soon) Использование кастомных запросов к базе данных для поиска снов по ключевым словам, местам, настроениям
 - (coming soon) Доступ к списку снов по паролю, сохраняемому в хешированном виде с помощью SharedPreferences
 - (coming soon) Смена языка
