# Осознанные сны [в разработке]
Мобильное приложение для ведения дневника снов и практики метода тестирования реальности. Построено 
по принципу single-activity architecture с использованием паттерна MVVM.
</br>
Особенность проекта: симуляция стратегии ветвления GitFlow с помощью создания дополнительного
[гит-аккаунта](https://github.com/kirillpolonskiismurf/LucidDreaming), выполняющего роль второго
участника команды.</br>
Использованные рабочие процессы:
 - Pull request
 - Симуляция code review с принятием и отказом PR

## Стек
 - Kotlin
 - View-based UI
 - Jetpack Navigation
 - Room
 - LiveData
 - Dagger Hilt
 - View binding
 - Coroutines & Flow
 - AlarmManager
## Возможности приложения
 - Создание, редактирование снов, их поиск с помощью различных фильтров
 - Настройка напоминаний (период активности, количество)
 - Выбор одной из 4 цветовых тем
 - (coming soon) Смена языка
## Реализованный функционал
 - Использование Navigation Component для переходов между фрагментами
 - Передача аргументов навигации с помощью плагина Safe Args
 - CRUD операции с помощью Room
 - Внедрение зависимостей в класс DreamRepository и различные View Models с помощью Dagger Hilt
 - Простое управление анимациями и пользовательскими касаниями в кастомных MotionLayout
 - Доступ к списку снов по паролю, сохраняемому в хешированном виде с помощью Preferences DataStore
 - Использование inexact alarms для триггера уведомлений
 - Использование статичного Broadcast Receiver для отправки уведомлений
 - Смена тем
 - (coming soon) Смена языка
